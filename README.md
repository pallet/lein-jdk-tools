# lein-jdk-tools

A Leiningen plugin to add `tools.jar` and `sa-jdi.jar` to the classpath.

## Usage

In your `project.clj` file, add `[lein-jdk-tools "0.1.1"]` into the
`:plugins` vector of the profile you want to have the jar files.
